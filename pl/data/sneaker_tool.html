- Typ: Aplikacja CLI + Backend
- Okres tworzenia: 2022/2023
- Język: Python
- Technologie: Web API, mySQL

Opis

Sneaker Tool to nazwa robocza dla zamówionej przez klienta kompleksowej aplikacji działającej dookoła tematu limitowanych i rzadkich produktów branży odzieżowej (głównie butów). Moim zadaniem było tworzenie, rozwój, testowanie i rozwiązywanie problemów zarówno aplikacji dla konsumentów jak i backendu. Zgodnie z wymogami użyłem do obu tych rzeczy języka Python. Do bazy danych odpowiadającej za dostęp do programu, limit dziennych użyć jego modułów oraz zbierania danych diagnostycznych użyłem mySQL. Na prośbę klienta nie upubliczniam szczegółów funkcjonalności programu, jednak w dużej mierze opierał się on na zapytaniach do zewnętrznych serwerów oraz przetwarzaniu zwracanych danych i wykonywaniu odpowiadającym im akcjom. Program niezłym zainteresowaniem i zbierał na tyle pochlebne opinie, że po około pół roku została zamówiona jego ulepszona wersja.
