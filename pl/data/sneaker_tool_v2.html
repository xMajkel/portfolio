- Typ: Aplikacja CLI + Backend
- Okres tworzenia: 2023/2024
- Język: GO
- Technologie: Web API, mySQL, Git

Opis

Sneaker Tool v2 to ulepszona wersja jego prekursora. Głównym powodem do odświeżenia programu była niska wydajność spowodowana językiem Python, gdyż jasnym okazało się, że od strony kodu nie da się wprowadzić usprawnień i natura języka interpretowanego ogranicza ogólną szybkość działania. Wybór padł na GO, z którym miałem już styczność i bardzo przypadł mi do gustu, szczególnie ze względu na dobrą obsługę wielowątkowości. Moje zadania pozostały takie same (tworzenie, rozwój, testowanie i naprawa błędów), dodatkowo zdecydowałem się również przenieść backend na GO. Przy okazji pisania programu zaznajomiłem się z oprogramowaniem Git do zarządzania wersjami oraz zmianami pomiędzy nimi występującymi.
